# Fluffy Math

> 🔣 四则运算 | BIT 软件工程

## 要求

### 第一阶段

- 一次生成 1000 道题目并保存至文件中（没有重复）🚩
- 表达式进行求值
- 除了整数，还要支持真分数四则运算
- 让程序支持接受用户输入答案，并判断对错，并给出汇总

### 第二阶段

- 增加「乘方」的运算符。

### 第三阶段

- 部署至服务器，实现在网页端的 SPA 程序。

## 编译 Building

1. 安装依赖

```bash
$ yarn install
```

2. 编译运行

```bash
$ yarn start
```
